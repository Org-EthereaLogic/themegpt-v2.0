## Key Recommendation

"If you're unsure which model to use, we recommend starting with **Claude Sonnet 4.5**. It offers the best balance of intelligence, speed, and cost for most use cases."

## Legacy Models Available

Older versions (Claude Sonnet 4, Claude Haiku 3.5, Claude Opus 4, etc.) remain accessible but migration to current models is advised for improved performance.

## Getting Started

Developers can access Claude through the Anthropic API, AWS Bedrock, and Google Vertex AI. All current models support text/image input, text output, and multilingual capabilities.
