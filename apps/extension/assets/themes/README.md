# Theme Screenshots

This directory contains preview screenshots for each theme. These are displayed in the extension popup when available.

## Screenshot Specifications

| Property | Value |
|----------|-------|
| **Format** | PNG (preferred) or WebP |
| **Dimensions** | 280 x 160 pixels |
| **Naming** | `{theme-id}.png` (e.g., `aurora-borealis.png`) |

## Capture Process

1. Open ChatGPT in Chrome with the extension loaded
2. Apply the target theme
3. Wait for any animated effects to be visible
4. Capture at the specified dimensions (280x160)
5. Save with the exact theme ID as filename

## Notes

- For animated themes, capture a frame that best represents the effect
- Ensure good contrast between UI elements
- Include a visible chat bubble and input field in frame
- The extension falls back to a mockup preview if screenshots are missing
