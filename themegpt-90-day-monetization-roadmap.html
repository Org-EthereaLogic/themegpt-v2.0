Current Observability Limits

GA4 and Clarity are consent-gated, so client-side funnels can undercount true commercial performance. This is expected behavior. Ad efficiency is measured from server-side events: checkout_sessions and revenue_events, both populated from Stripe session creation and webhook callbacks.

Resolved Implementation Work

  • Checkout regression fixed by removing consent_collection.promotions from Stripe Checkout session creation.
  • First-party attribution capture implemented from URL params: utm_source, utm_medium, utm_campaign, utm_content, utm_term, gclid, fbclid, ttclid.
  • Attribution now propagates end-to-end: browser landing params → checkout API payload → Stripe metadata → webhook revenue records.
  • GA4 funnel and intent events now include: pricing_view, theme_preview, checkout_abandon, checkout_start, purchase_success, trial_start.
  • Internal monetization endpoint added: /api/metrics/monetization?days=7 (restricted to internal accounts) for server-side KPI tracking.
  • Checkout regression guard added: apps/web/app/api/checkout/route.test.ts verifies checkout creation does not set consent_collection.promotions.
  • Microsoft Clarity integrated via @microsoft/clarity npm SDK, consent-gated alongside GA4 using the same themegpt_analytics_consent localStorage key.
  • Clarity linked to GA4 property 516189580; two conversion funnels configured (Marketing → Purchase, Extension → Purchase).
  • CWS GA4 opted in; install event marked as key event with $1 proxy value for ROAS calculations.

Active Guardrails (Paid Launch)

Control Current state Trigger / rule Action
Channel scope Start with 2 channels only. Google Search + Reddit paid social. Pause adding channels until first attributed revenue event.
Daily spend cap Applies from launch day. $25/day/channel hard cap. Do not increase caps during first 7 days.
No-signal kill switch Active. >= $75 spend on a channel with zero checkout sessions. Pause the channel immediately and rotate creatives.
CAC kill switch Active after first 2 purchases/channel. CAC exceeds $45 for 3 consecutive days. Pause paid traffic for that channel and revise offer/landing.
Weekly geography watch India and US currently near parity in CWS baseline. India share ties/exceeds US for 2 consecutive weekly scorecards. Shift ad geo targeting toward US/UK/CA/AU and re-check copy.
