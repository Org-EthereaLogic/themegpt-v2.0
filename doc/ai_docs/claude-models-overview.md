# Claude Models Overview

## Recommended Starting Point

Anthropic suggests beginning with **Claude Sonnet 4.5**, which provides "the best balance of intelligence, speed, and cost for most use cases, with exceptional performance in coding and agentic tasks."

## Current Model Lineup

The latest Claude offerings include four primary models:

**Claude Sonnet 4.5** - Positioned as the smartest model for complex agents and coding work
- API ID: claude-sonnet-4-5-20250929
- Pricing: $3/input million tokens, $15/output million tokens
- Context: 200K tokens standard, 1M tokens in beta
- Max output: 64K tokens
- Knowledge cutoff: January 2025

**Claude Haiku 4.5** - The fastest model with near-frontier capabilities
- API ID: claude-haiku-4-5-20251001
- Pricing: $1/input million tokens, $5/output million tokens
- Context: 200K tokens
- Max output: 64K tokens
- Knowledge cutoff: February 2025

**Claude Opus 4.5** - Premium model combining maximum intelligence with practical performance
- API ID: claude-opus-4-5-20251101
- Pricing: $5/input million tokens, $25/output million tokens
- Max output: 64K tokens
- Knowledge cutoff: March 2025

**Claude Opus 4.1** - Specialized for reasoning tasks
- API ID: claude-opus-4-1-20250805
- Pricing: $15/input million tokens, $75/output million tokens
- Max output: 32K tokens

## Key Features

All current models support:
- Text and image input
- Text output
- Multilingual capabilities
- Vision processing
- Extended thinking
- Priority tier access

Models are accessible through the Anthropic API, AWS Bedrock, and Google Vertex AI.

## Legacy Models Available

Older versions including Claude Sonnet 4, Claude Sonnet 3.7, Claude Opus 4, and Claude Haiku variants remain accessible but users are encouraged to upgrade.

## Next Steps

Developers can explore Claude through the [quickstart guide](/docs/en/get-started) or test prompts directly in the Claude Console.
